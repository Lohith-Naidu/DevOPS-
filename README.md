# 🖥️ System Health Monitoring Dashboard

## 🚀 Overview

A real-time system monitoring project that tracks CPU, Memory, and Disk usage and provides instant alerts when system performance becomes critical.

This project is designed as a **portfolio showcase project** demonstrating automation, monitoring, and alerting concepts used in DevOps systems.

---

## 🎯 Key Features

- 📊 Real-time system monitoring (CPU, Memory, Disk)
- 🚦 Automatic health status detection (OK / WARN / CRITICAL)
- 📲 Instant Telegram alerts for system failures
- ⏱ Automated execution using scheduled system jobs
- 🧾 Logging and activity tracking

---

## 🧠 How It Works

1. System continuously monitors resource usage  
2. It evaluates system health based on thresholds  
3. If system becomes critical, alert is triggered  
4. Logs are stored for tracking and debugging
